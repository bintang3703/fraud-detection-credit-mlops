This app helps you review credit card transactions and spot possible fraud.  
It uses a machine learning pipeline with LightGBM, SMOTE, SHAP, KS, Gini, and PSI.

You can use it to:

- Check transaction risk
- Review model results in a simple dashboard
- See which factors influence a prediction
- Compare model quality with clear metrics
- Explore data in a visual way

The app runs in Streamlit, so the interface is simple and easy to use.

## 📊 Main features

### 🧾 Transaction review

Review transaction data and check which items look unusual.  
This helps you focus on records that may need a closer look.

### 🧠 Model scoring

See how the model scores each transaction.  
The app uses LightGBM to classify records and support fraud review.

### 🎯 Imbalanced data handling

Fraud data often has far fewer fraud cases than normal cases.  
The app uses SMOTE to help balance the training data so the model can learn better.

### 🔍 Explainable results

Use SHAP to see why the model made a prediction.  
This helps you understand the main factors behind each result.

### 📈 Model checks

The app includes common quality measures:

- KS
- Gini
- PSI

These metrics help you see how the model performs and whether the data has changed.

### 📋 Interactive dashboard

The Streamlit dashboard lets you explore results without special tools.  
You can move through the app with buttons, tables, and charts.
